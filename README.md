# Smart Retail POS

A lightweight **Point of Sale (POS)** system built with **Next.js and MongoDB** designed for small retail businesses.  
This system is optimized for **single-shop operations with 1–2 staff members**, enabling efficient management of inventory, billing, and daily sales.

---

## 🚀 Features

- 🧾 Create and manage invoices
- 📦 Inventory and product management
- 📊 Track daily sales
- 💳 Multiple payment methods (cash, card, etc.)
- 🔎 Product search and quick billing
- 🛠 Simple and intuitive interface for shop staff
- ☁️ Cloud deployment with Vercel
- 🗄 MongoDB database for flexible data storage

---

## 🏗 Tech Stack

**Frontend**
- Next.js
- React
- TailwindCSS (optional)

**Backend**
- Next.js API Routes

**Database**
- MongoDB Atlas

**Deployment**
- Vercel
